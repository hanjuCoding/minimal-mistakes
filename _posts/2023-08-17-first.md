## 1000번

## 10430번

## 10171번
\, ' 등의 문자에 주의하며 고양이를 출력하는 문제
```python
print("\\    /\\")
print(" )  ( \')")
print("(  /  )")
print(" \\(__)|")
```
\이랑' 을 출력하려면 앞에 \를 붙혀줘야한다.  

## 10172번
", `, \ 등의 문자에 주의하며 개를 출력하는 문제
```python
print("|\\_/|")
print("|q p|   /}")
print("( 0 )\"\"\"\\")
print("|\"^\"`    |")
print("||_/=\\\\__|")
```
"또한 마찬가지로 앞에\를 붙혀줘야한다.
## raw string 이용하기 
문자열 앞에 r을 하나 붙히면 escape문(\b,\",\n)이 동작되지 않음.
```python
#ex) 10172번 마지막 줄
print(r"||_/=\\__|")
```
