## 1330번
두 수 비교하기
```python

    print("<")

    print("==")
````

## 2525번
오븐 시계
```python
A, B = map(int,input().split())
C = int(input())
#C를 시간, 분단위로 쪼개서 A,B에 나눠주기
A += C//60
B += C%60

#B가 60이상일때 다시 시간,분 단위로 쪼개 A,B에 나눠주기
if B >= 60:
    A += B//60
    B = B%60
if A > 23: #A가 24시를 초과할 경우
    A -= 24
    
print(A,B)
````

## 2480번
주사위 세개
```python
A, B, C = map(int,input().split())
total = 0
if A == B == C:
    total += 10000 + A*1000
elif A == B != C or A == C != B :
    total += 1000+A*100
elif  B == C != A:
    total += 1000+B*100
else:
    total += max(A,B,C)*100

print(total)
````
