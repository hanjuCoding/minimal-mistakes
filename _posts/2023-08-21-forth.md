# 파이썬list


## 10807번  
배열을 입력받고 v를 찾는 문제
```python
N = int(input())
arr = list(map(int,input().split()))
v = int(input())
count = 0
for i in range(N):
    if arr[i] == v:
        count +=1

print(count)

````

## 10871번  
배열을 입력받고 X보다 작은 수를 찾는 문제
```python
N,X = map(int,input().split())
A = list(map(int,input().split()))
for i in range(N):
    if A[i] < X :
        print(A[i],end=" ")
````

## 10818번  
최솟값과 최댓값을 찾는 문제
```python
N = int(input())
arr = list(map(int,input().split()))


for i in range(N):
    if i == 0:
        min = arr[i]
        #min_count = i+1
    if i != N-1:
        if min > arr[i+1]:
            min = arr[i+1]
            #min_count = i+2

for i in range(N):
    if i == 0:
        max = arr[i]
        #max_count = i+1
    if i != N-1:
        if max < arr[i+1]:
            max = arr[i+1]
            #max_count = i+2
    
print(min,max)
````
